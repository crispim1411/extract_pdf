Configuração do Ambiente do Projeto
************************************

EXplicar oq faz

Dependências
=============
* Python 3.7+
  - $ python --version
* MongoDB
  - $ mongod --version

Procedimento de setup
======================
#. Instalar dependências do Python:

    * $ ./config.sh

#. Ativar ambiente virtual:

    * $ source venv/bin/activate

#. Subir instância do MongoDB:

    * $ mongod

Utilização
===========
* $ python
